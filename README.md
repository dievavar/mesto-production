# Mesto

Проект фронтенд-приложения с публикацией через GitHub Pages.

Ссылка на проект: https://dievavar.github.io/mesto-production/

---

## Публикация проекта

Проект автоматически публикуется через GitHub Actions при каждом `push` в ветку `main` (npm run deploy)
